# Expense Tracker App

A simple and intuitive Android application built to help users manage their personal finances by tracking income, expenses, and account balances. This project was developed to solve a personal need for monitoring money flow and gaining better insights into spending habits.

## Features

* 📊 **Track Money Flow**

  * Record income and expense transactions.
  * Maintain a complete history of financial activities.

* 💼 **Multiple Accounts**

  * Create and manage multiple accounts such as Cash, Bank, Wallet, or Savings.
  * Track balances for each account independently.

* 📈 **Financial Analysis**

  * Analyze transactions using various filters.
  * View spending and income based on selected criteria such as date, account, category, and transaction type.
  * Gain insights into financial trends and money management.

* 🔍 **Advanced Filtering**

  * Filter transaction records to quickly find specific entries.
  * Simplifies reviewing expenses and monitoring budgets.

* 📝 **Easy Transaction Management**

  * Add, edit, and delete transactions with a clean and user-friendly interface.

* 📱 **Modern Android UI**

  * Designed with a focus on simplicity, usability, and efficient navigation.

## Tech Stack

* **Language:** Java
* **IDE:** Android Studio
* **Database:** SQLite
* **Platform:** Android

## Motivation

This application was created primarily for personal use to gain better control over daily finances. Existing expense tracking applications often contained unnecessary features or subscription requirements, so this project was built with simplicity, customization, and practical usability in mind.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/AbhijitMohanty151/Expense-tracker-app.git
```

2. Open the project in Android Studio.

3. Sync the Gradle files.

4. Build and run the application on an Android device or emulator.
